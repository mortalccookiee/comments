# comments

这个仓库用于课程作业或课程反馈收集，主要通过 Pull Request 的方式让不同同学提交总结、评价、意见和建议。

## 主要内容

- `group00.md` 到 `group35.md`：不同分组或同学提交的课程总结与评论。
- `merge-pr.qmd`：与合并 Pull Request 或整理评论相关的 Quarto 文档。
- `.github/`：GitHub 仓库配置。

## 使用方式

提交评论时通常通过新建分支和 Pull Request 完成。提交内容中需要保留学号，方便识别和评分。

## 回顾时怎么看

这个仓库不是代码项目，主要用于回顾课程反馈、协作提交过程和 Pull Request 练习。

## 注意事项

- 不要随意删除其他人的评论内容。
- 如果需要补交或修改，优先在原 Pull Request 上继续提交。
